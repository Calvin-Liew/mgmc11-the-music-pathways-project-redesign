# The Music Pathways Project - Landing Page

A modern, responsive landing page for The Music Pathways Project (TMPP) at the University of Toronto Scarborough.

## Features

- **React + TypeScript + Tailwind CSS**
- Fully responsive (mobile-first design)
- Smooth scroll navigation
- Interactive tabs for audience sections
- Accessible semantic HTML
- Modern UI with hover effects and transitions

## Getting Started

### Prerequisites

- Node.js 18+ and npm/yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Project Structure

- `MusicPathwaysLandingPage.tsx` - Main landing page component
- `src/App.tsx` - App entry point
- `src/main.tsx` - React DOM root
- `src/index.css` - Tailwind CSS imports
- `tailwind.config.js` - Tailwind configuration
- `vite.config.ts` - Vite configuration

## Customization

All copy and content is defined in the `COPY` constant at the top of `MusicPathwaysLandingPage.tsx`. Brand colors are defined in the `COLORS` constant.
